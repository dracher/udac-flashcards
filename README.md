# This is a react-native-app for UDA class Project 3

## Usage

1. clone the repo to local

```
$> git clone ...
```

2. install the dep packages, use `npm` or `yarn`

```
$> cd $PROJECT_DIR && yarn
```

3. lanuch the project

```
$> yarn start
```

4. (optional) if u meet `rlimit` issue, then run with 

```
$> make start
```
